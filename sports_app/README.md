# Лабораторная работа №2 (часть 2)
**Курс:** «Проектирование программного обеспечения интеллектуальных систем»

## Вариант 7 — «Спортсмены»
Поля записи:
- ФИО спортсмена (строка)
- Состав: `основной / запасной / n/a`
- Позиция (строка)
- Титулы (целое число)
- Вид спорта (строка)
- Разряд: `1-й юношеский / 2-й разряд / 3-й разряд / кмс / мастер спорта`

Условия поиска/удаления (как в варианте):
- по ФИО **или** виду спорта;
- по количеству титулов (нижний/верхний предел);
- по ФИО **или** разряду.

> В диалоге поиска списки «Вид спорта» и «Разряд» собираются системой из текущих данных БД и показываются в выпадающих списках.

---

## Архитектура и требования
- GUI на **tkinter/ttk**
- Архитектура **MVC**:
  - `app/model/*` — модель + SQLite репозиторий + XML IO
  - `app/view/*` — окна/диалоги/виджеты
  - `app/controller/*` — контроллер, связывает View и Model
- Хранение записей в **SQLite** (файл `.db`)
- Сохранение в XML через **DOM (minidom)**
- Загрузка из XML через **SAX (xml.sax)**
- Постраничный вывод (по умолчанию 10): первая/пред/след/последняя, выбор количества на странице, отображение текущей страницы и общего числа записей
- Альтернативное отображение массива как **дерево**, где листовые элементы соответствуют полям записи
- Меню + панель инструментов (дублируют команды)

---

## Запуск
Требования: Python 3.10+

```bash
python run.py
```

По умолчанию БД создаётся в `data/app.db`.

---

## Сценарий демонстрации (пример)
1. Запустить приложение.
2. `Файл → Загрузить из XML…` и выбрать один из файлов из `samples/` (в каждом ≥50 записей).
3. Убедиться, что записи появились в главном окне (вкладка «Таблица») и на вкладке «Дерево».
4. Открыть `Данные → Поиск…`:
   - выбрать «По ФИО или виду спорта», указать часть ФИО **или** выбрать вид спорта из списка → посмотреть постраничные результаты.
   - выбрать «По количеству титулов», задать нижний и верхний предел.
   - выбрать «По ФИО или разряду», указать часть ФИО **или** разряд.
5. Открыть `Данные → Удалить…`, удалить по одному из условий и увидеть сообщение «сколько удалено».
6. `Файл → Сохранить в XML…` — экспорт текущего массива в XML.
7. (Опционально) `Файл → Выбрать/создать БД…` — переключить SQLite-файл.
